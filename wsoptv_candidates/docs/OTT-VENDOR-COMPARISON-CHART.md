## 💰 비용 비교 (실제 견적 기반)

### 3년 총 비용 (TCO)

| 항목 | Vimeo | Brightcove | **메가존** | 맑음소프트 |
|------|------:|----------:|-----------:|----------:|
| **구축비** | 미제출 | 미제출 | **50억** | 1~2억 |
| **연간 운영비** | 미제출 | 미제출 | **10~15억** | 2~3억 |
| **3년 운영비** | 미제출 | 미제출 | **30~45억** | 6~9억 |
| **3년 TCO** | **미제출** | **미제출** | **80~100억** | **7~11억** |

> **메가존 실제 견적**: 구축비 **50억 (3년, 운영비 제외)**
>
> **운영비 추산**: TVING 운영비(~50억/년)의 **20~30% 적용** → **10~15억/년**
> - WSOPTV는 TVING 대비 규모가 작음 (MAU ~2%, 콘텐츠 ~5%)
> - 초기 인프라 고정비용 고려하여 보수적 적용
>
> **3년 TCO 계산**: 구축비 50억 + 운영비 10~15억/년 × 3년 = **80~100억**
>
> ⚠️ **Vimeo, Brightcove**: 견적 미제출 상태로 비용 비교 불가

### 비용 상세 내역

| 비용 항목 | Vimeo | Brightcove | 메가존 | 맑음소프트 |
|----------|-------|------------|--------|------------|
| **플랫폼 라이선스** | 미제출 | 미제출 | 포함 | 월 50~100만 |
| **앱 개발** | 미제출 | 미제출 | **별도 개발** | 포함 (제한적) |
| **CDN** | 미제출 | 미제출 | AWS CloudFront | 포함 |
| **DRM** | 미제출 | 미제출 | AWS 기반 구현 | 포함 |
| **기술 지원** | 미제출 | 미제출 | **전담팀** | 전담팀 |
| **커스터마이징** | 불가 | 제한적 | **무제한** | 제한적 |

### 메가존 비용 계산 (TVING 벤치마크 기반, 규모 보정)

| 항목 | 금액 | 산정 근거 |
|------|-----:|----------|
| **구축비 (3년)** | **50억** | 메가존 견적 (운영비 제외) |
| **연간 운영비** | **10~15억** | TVING의 20~30% (규모 차이 반영) |
| **3년 운영비** | 30~45억 | 10~15억 × 3년 |
| **3년 TCO** | **80~100억** | 구축비 + 3년 운영비 |

#### 규모 비교 (TVING vs WSOPTV)

| 항목 | TVING | WSOPTV | 비율 |
|------|------:|-------:|-----:|
| MAU | 500만+ | 5~10만 | ~2% |
| 콘텐츠 | 수만 편 | 수백~천 편 | ~5% |
| 동시접속 | 수십만 | 수천~만 | ~5% |

> **보수적 20~30% 적용**: 초기 인프라 고정비용, 기본 인력 등 고려
>
> ⚠️ **Vimeo, Brightcove**: 피드백 느림, 견적 미제출로 비용 비교 불가

```
┌─────────────────────────────────────────────────────────┐
│                  3년 총 비용: 80~100억                   │
├─────────────────────────────────────────────────────────┤
│  구축비: 50억 (3년)        운영비: 10~15억/년 × 3년     │
│  (운영비 제외)             = 30~45억 (TVING의 20~30%)   │
├─────────────────────────────────────────────────────────┤
│  ✅ Multi-view (2x2)         ✅ Timeshift/DVR           │
│  ✅ Player Cam Mode          ✅ GGPass SSO              │
│  ✅ Featured Hands           ✅ Samsung/LG TV 앱        │

└─────────────────────────────────────────────────────────┘

- 구축비: 메가존 견적 50억 (3년, 운영비 제외)
- 운영비: TVING(~50억/년)의 20~30% = 10~15억/년
  * WSOPTV 규모: MAU ~2%, 콘텐츠 ~5%, 동시접속 ~5%
  * 보수적 비율 적용 (초기 고정비용 고려)
```

## 한국 OTT 운영비 벤치마크

| OTT 서비스 | 연간 운영비 | MAU | 비고 |
|-----------|----------:|----:|------|
| **TVING** | **~50억** (인프라) | 500만+ | 콘텐츠 투자 별도 2,300억 |
| **웨이브** | ~100억+ | 400만+ | 손실 1,217억 (2022) |
| **왓챠** | ~50억+ | 100만+ | 손실 555억 (2022) |
| **WSOPTV (메가존)** | **10~15억 (추정)** | 5~10만 | TVING의 20~30% 적용 |

> **운영비 산정 근거**:
> - TVING 연간 운영비 ~50억 기준
> - WSOPTV는 MAU ~2%, 콘텐츠 ~5% 규모 → **20~30% 비율 적용**
> - 초기 인프라 고정비용 고려하여 보수적 산정
> - WSOPTV는 기존 WSOP 콘텐츠 활용으로 콘텐츠 비용 최소화 가능
