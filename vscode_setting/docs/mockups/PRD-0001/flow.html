Multi-VSCode Launcher Flow

STEP 1
Start Script
vscode-launcher.ps1 실행
./vscode-launcher.ps1
↓
STEP 2
Load Configuration
JSON 설정 파일에서 프로젝트 목록 로드
vscode-projects.json
↓
STEP 3
Calculate Window Layout
화면 해상도 기반 그리드 레이아웃 계산
↓
STEP 4a
Launch VSCode
각 프로젝트별 IDE 실행
code --new-window [path]
STEP 4b
Position Windows
창 위치 및 크기 자동 배치
↓
STEP 5
Execute Claude Command
각 터미널에서 Claude 자동 실행
claude --dangerously-skip-permissions
↓
STEP 6
Monitor Status
실행 상태 모니터링 및 로깅
