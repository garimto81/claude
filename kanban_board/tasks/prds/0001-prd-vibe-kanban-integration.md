# PRD-0001: Vibe Kanban 통합 - Claude Code 프로젝트 칸반 보드

**버전**: 1.0.0
**작성일**: 2026-01-14
**상태**: Draft
**우선순위**: High

---

## 1. 개요

### 1.1 목적

Claude Code로 작업 중인 다수의 프로젝트들을 칸반 보드 형식으로 시각화하고, 작업을 효율적으로 관리 및 처리할 수 있는 통합 환경을 구축한다.

### 1.2 범위

- Vibe Kanban 설치 및 초기 설정
- `C:\claude` 모노레포 내 프로젝트들 연동
- Claude Code 에이전트 기반 작업 자동화
- GitHub PR 워크플로우 통합

### 1.3 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 멀티 프로젝트 보드 | 여러 프로젝트를 하나의 칸반 뷰로 관리 | P0 |
| 병렬 에이전트 실행 | 여러 작업을 동시에 Claude Code로 처리 | P0 |
| GitHub 연동 | PR 자동 생성, 머지, 리베이스 | P1 |
| 실시간 진행 모니터링 | 작업 상태 및 로그 실시간 확인 | P1 |
| VS Code 확장 | IDE 내 칸반 보드 접근 | P2 |

## 3. 기술 스택

| 구분 | 기술 | 버전 | 용도 |
|------|------|------|------|
| **런타임** | Node.js | 18+ | Vibe Kanban 실행 |
| **백엔드** | Rust | stable | Vibe Kanban 코어 |
| **프론트엔드** | Web UI | - | 칸반 보드 인터페이스 |
| **에이전트** | Claude Code | latest | AI 코딩 자동화 |
| **버전관리** | Git Worktrees | - | 작업 격리 |
| **연동** | GitHub API | v4 | PR 자동화 |

## 4. 구현 단계

### Phase 1: 환경 설정 (Day 1)

- [ ] Node.js 18+ 설치 확인
- [ ] Claude Code CLI 인증 확인
- [ ] `npx vibe-kanban` 설치 및 실행
- [ ] 웹 UI 접속 확인 (localhost:5173)

### Phase 2: 프로젝트 연동 (Day 1-2)

- [ ] `C:\claude` 모노레포 등록
- [ ] 주요 프로젝트 보드 생성:
  - `gfx_json` - GFX JSON 동기화
  - `youtuber_chatbot` - YouTube 챗봇
  - `automation_ae` - AE 자동화
  - `iconik_parser` - Iconik 파서
  - `mad_framework` - MAD 프레임워크
- [ ] Claude Code 에이전트 설정

### Phase 3: 워크플로우 설정 (Day 2)

- [ ] 칸반 컬럼 구성:
  - Backlog: 미정렬 작업
  - Ready: 준비된 작업
  - In Progress: 진행 중 (에이전트 실행)
  - Review: PR 리뷰 대기
  - Done: 완료
- [ ] GitHub 연동 설정
- [ ] PR 자동 생성 규칙 설정

### Phase 4: 확장 설정 (Day 3)

- [ ] VS Code 확장 설치 (선택)
- [ ] MCP 통합 설정 (선택)
- [ ] 알림 설정 (Slack/Discord)
