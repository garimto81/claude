# 디자인 시스템 스타일 검증

이 문서는 Google Workspace Skill v2.3.2의 스타일 시스템을 검증합니다.

## 타이포그래피 테스트

### H3 헤딩 (12pt, Bold, #404040)

#### H4 헤딩 (11pt, SemiBold, #404040)

##### H5 헤딩 (11pt, SemiBold, #404040)

###### H6 헤딩 (10pt, SemiBold, #666666)

본문 텍스트입니다. 11pt, Regular, #404040 색상이 적용되어야 합니다.

## 테이블 스타일 테스트

| 요소 | 크기 | 색상 |
|------|------|------|
| Title | 26pt | `#1A4D8C` |
| H1 | 18pt | `#1A4D8C` |
| H2 | 14pt | `#3373B3` |
| **볼드 텍스트** | 11pt | #404040 |
| *이탤릭 텍스트* | 11pt | #404040 |

## 인라인 스타일 테스트

일반 텍스트와 **볼드**, *이탤릭*, `인라인 코드` 테스트입니다.

**볼드**와 *이탤릭*이 함께 있는 문장입니다.

## 코드 블록 테스트

```python
# 코드 블록 (10.5pt, Consolas, 배경 #F2F2F2)
def hello():
    print("Hello, World!")
```

## 강조 색상 참조

| 색상 | HEX | 용도 |
|------|-----|------|
| Red | `#DC2626` | 오류 |
| Green | `#059669` | 성공 |
| Blue | `#1A4D8C` | 정보 |
| Yellow | `#CA8A04` | 주의 |

## 리스트 테스트

- 불릿 리스트 항목 1
- 불릿 리스트 항목 2
- **볼드** 항목과 `코드` 포함

> 인용문 테스트입니다. 왼쪽에 파란색 테두리가 표시되어야 합니다.
