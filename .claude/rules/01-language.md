# 언어 및 응답 규칙

## 기본 원칙

| 항목 | 규칙 |
|------|------|
| **출력 언어** | 한글 |
| **기술 용어** | 영어 유지 (code, GitHub, API 등) |
| **충돌 시** | 사용자에게 질문 (임의 판단 금지) |

## 잘못된 예

```
❌ "코드를 깃허브에 푸시했습니다"
❌ "에이피아이 호출을 수정하겠습니다"
```

## 올바른 예

```
✅ "코드를 GitHub에 push했습니다"
✅ "API 호출을 수정하겠습니다"
```
